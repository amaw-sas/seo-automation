## 🎨 Cambios Técnicos Implementados

### 1. Modificación de ImportKeywords

**Cambio**: Soporte para archivos XLSX además de CSV

**Antes**:
```php
$csvFiles = glob($sourceDir . '/*.csv');
// Solo procesaba CSVs con League\Csv
```

**Después**:
```php
$csvFiles = glob($sourceDir . '/*.csv');
$xlsxFiles = glob($sourceDir . '/*.xlsx');
$allFiles = array_merge($csvFiles, $xlsxFiles);

// Detecta extensión y usa PhpSpreadsheet para XLSX
if ($extension === 'xlsx') {
    $spreadsheet = IOFactory::load($file);
    $sheet = $spreadsheet->getActiveSheet();
    $rows = $sheet->toArray();
    // Convierte a formato asociativo
}
```

**Resultado**: +72,257 keywords procesadas (de las cuales 59,191 únicas).

### 2. Modificación de ImportRankings

**Cambio**: Soporte para archivos `keywords.xlsx` de competidores (no solo `organic-keywords.xlsx`)

**Antes**:
```php
$possiblePaths = [
    $baseDir . '/mis-dominios/' . $domainSlug . '/organic-keywords.xlsx',
    $baseDir . '/competidores/' . $domainSlug . '/organic-keywords.xlsx',
```

**Después**:
```php
$possiblePaths = [
    $baseDir . '/mis-dominios/' . $domainSlug . '/organic-keywords.xlsx',
    $baseDir . '/competidores/' . $domainSlug . '/organic-keywords.xlsx',
    $baseDir . '/competidores/' . $domainSlug . '/keywords.xlsx', // Nuevo
];
```

## ⚠️ Issues Identificados

### 1. Dominios Propios - Rankings Duplicados
**Issue**: Rankings previos fueron re-importados con nueva snapshot_date (2026-02-07), creando duplicados.

**Evidencia**:
- alquilatucarro: 498 rankings (antes: 130) = +368
- alquilame: 592 rankings (antes: 100) = +492
- alquicarros: 465 rankings (antes: 100) = +365

**Causa**: El comando usa `snapshot_date` como unique key, pero se ejecutó con fecha diferente.

**Impacto**: ⚠️ Medio - Los datos antiguos siguen ahí, pero ahora hay 2 snapshots por dominio.

**Solución recomendada**: Limpiar rankings antiguos o usar `--snapshot-date` consistente.

### 2. Despegar.com.co - Sin Datos
**Issue**: No se encontró archivo de keywords para despegar.com.co.

**Evidencia**: 0 rankings importados.

**Causa**: El archivo no existe en `../semrushdiego/competidores/despegar/`.

**Impacto**: ℹ️ Bajo - Despegar es un agregador, no competidor directo.

**Acción**: Ninguna necesaria por ahora.

### 3. Keywords Sin Ciudad - 91%
**Issue**: 53,724 keywords (91%) no tienen ciudad asignada.

**Causa**: Keywords genéricas ("alquiler de carros", "rent a car") o de otros temas (turismo, viajes, agencias).

**Impacto**: ℹ️ Bajo - Es esperado. No todas las keywords son geo-específicas.

**Acción**: Ninguna necesaria. El 9% con ciudad es suficiente para análisis local.
